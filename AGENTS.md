# Instructions pour Codex

Projet pédagogique simple en HTML, CSS et JavaScript vanilla.

Règles :
- Ne pas ajouter de dépendance npm.
- Créer du code lisible, simple et commenté seulement quand utile.
- Préférer des changements petits et vérifiables.
- Après modification, expliquer les fichiers changés.
- Ne pas modifier de fichiers hors du projet.
